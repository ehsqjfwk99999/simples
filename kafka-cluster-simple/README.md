kafka-cluster-simple
====================

Simple Apache Kafka cluster.

Powered by ...
--------------
- Docker Compose (`docker-compose.yaml`)

Connect Kafka
-------------
- `localhost:9092`
- `localhost:9093`
- `localhost:9094`

Connect Zookeeper
-----------------
- `localhost:2181`
- `localhost:2182`
- `localhost:2183`

Connect ZooNavigator
--------------------
```
localhost:9000
```
